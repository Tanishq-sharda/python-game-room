# Python Game Room

A console-based Python game platform where users can play multiple mini-games against the computer and accumulate points.

## Description

This project implements a text-based game room with three games:

- Number Guess
- Modified Rock Paper Scissors Lizard Spock
- Coin Flip

The program allows the user to select games from a menu, play multiple rounds, and track a running score until they choose to quit.

## Features

- Menu-driven command-line interface
- Score tracking across multiple rounds
- Randomized game outcomes
- Input handling for multiple game selections

## Technologies

- Python
- Functions
- Control flow
- Random number generation
- Command-line interaction
